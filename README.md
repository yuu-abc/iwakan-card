# 違和感カード

GitHub Pages でそのまま公開できる、静的なカード型メモアプリです。ChatGPT などで整えた JSON を貼り付けて保存し、「違和感」「表向きの説明」「隠れた前提」「本当の論点」「線引き」「次に見るもの」をカードとして見返せます。

## 使い方

1. `index.html` をブラウザで開きます。
2. 画面右側のプロンプトをコピーし、自由メモを ChatGPT などで JSON に整えます。
3. 生成された JSON を画面左側の入力欄に貼り付けます。
4. 必要に応じて画像 URL を JSON に入れるか、「画像を選択」から dataUrl として追加します。
5. 「カードとして保存」を押すと、ブラウザの `localStorage` に保存されます。

## 主な機能

- JSON から違和感カードを保存
- `id` / `createdAt` / `updatedAt` の自動付与
- カード一覧表示、編集、削除
- お気に入り登録とお気に入りのみ表示
- カード内容の折りたたみ / 展開
- キーワード、カテゴリ、タグによる絞り込み
- 画像 URL と dataUrl 画像の表示
- 保存済みカードの JSON エクスポート / インポート
- 初回表示用のサンプルカード

## GitHub Pages で公開する方法

1. このリポジトリを GitHub に push します。
2. GitHub のリポジトリ画面で **Settings** → **Pages** を開きます。
3. Source で公開したいブランチを選び、フォルダは `/root` を指定します。
4. 表示された URL にアクセスするとアプリを使えます。

## データ保存について

このアプリはサーバーや API を使いません。カードは利用者のブラウザ内の `localStorage` に保存されます。別のブラウザや端末では共有されないため、必要に応じて「JSONエクスポート」でバックアップしてください。

画像を dataUrl として保存すると `localStorage` の容量を多く使います。大きい画像や多数の画像を保存する場合は、画像 URL の利用も検討してください。
