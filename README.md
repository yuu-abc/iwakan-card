# 違和感カード

GitHub Pages でそのまま公開できる、静的なカード型メモアプリです。ChatGPT などで整えた JSON を貼り付けて保存し、「違和感」「表向きの説明」「隠れた前提」「本当の論点」「線引き」「次に見るもの」をカードとして見返せます。

## 使い方

画面上部にも使い方ガイドを表示しています。初めて使う場合は、次の流れで進めてください。

1. `index.html` をブラウザで開きます。
2. 最初の入力欄に、気になった出来事や言葉を自由メモとしてそのまま書きます。
   - 例：`上司が“自由に意見して”と言ったのに、反対意見だけ流された気がした。何が引っかかったのか整理したい。`
3. 「依頼文を作成してコピー」を押します。ChatGPT に貼り付けるための依頼文が自動で作られ、クリップボードにコピーされます。
4. ChatGPT に依頼文を貼り付けます。返ってきたJSONだけを、次の入力欄に貼り付けます。
5. JSONの形式・保存されるカード数が画面に表示されるので、問題なければ「カードとして保存」を押します。
6. 保存したカードはブラウザの `localStorage` に保存され、検索・カテゴリ絞り込み・お気に入りで見返せます。入力途中のJSONは下書きとして残ります。

### ChatGPTから返ってくるJSONの例

```json
[
  {
    "title": "自由に意見してと言われた会議",
    "category": "仕事",
    "tags": ["会議", "心理的安全性"],
    "memo": "上司が“自由に意見して”と言ったのに、反対意見だけ流された気がした。",
    "iwakan": "自由と言いながら、歓迎される意見が決まっているように感じた。",
    "surfaceExplanation": "時間がなかったので流しただけかもしれない。",
    "hiddenAssumption": "反対意見は場を乱すものだという前提があるかもしれない。",
    "realIssue": "安心して異論を出せる場になっているか。",
    "boundary": "発言を促すなら、異論にも応答する必要がある。",
    "nextWatch": "次回、反対意見への反応を観察する。"
  }
]
```

## 主な機能

- JSON から違和感カードを保存
- 3ステップの保存導線と、保存前のJSONチェック
- 自由メモからChatGPT向けJSON整形依頼文を自動生成・コピー
- 入力中JSONの下書き保存
- `id` / `createdAt` / `updatedAt` の自動付与
- カード一覧表示、編集、削除
- お気に入り登録とお気に入りのみ表示
- カード内容の折りたたみ / 展開
- キーワード、カテゴリ、タグによる絞り込み
- 画像 URL と dataUrl 画像の表示
- 保存済みカードの JSON エクスポート / インポート
- 初回表示用のサンプルカード

## GitHub Pages で公開する方法

1. このリポジトリを GitHub に push します。
2. GitHub のリポジトリ画面で **Settings** → **Pages** を開きます。
3. Source で公開したいブランチを選び、フォルダは `/root` を指定します。
4. 表示された URL にアクセスするとアプリを使えます。

## データ保存について

このアプリはサーバーや API を使いません。カードは利用者のブラウザ内の `localStorage` に保存されます。別のブラウザや端末では共有されないため、必要に応じて「JSONエクスポート」でバックアップしてください。

画像を dataUrl として保存すると `localStorage` の容量を多く使います。大きい画像や多数の画像を保存する場合は、画像 URL の利用も検討してください。
